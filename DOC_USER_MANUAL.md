# 喵选（Meowpick）小程序使用指南

本指南旨在帮助测试人员、产品经理或最终用户快速了解和上手目前最新版本（包含提议功能）的喵选小程序。

## 1. 核心概念：吐槽 vs 提议
*   **吐槽（Comment）**：针对**数据库中已存在且收录**的课程发表的评价、打分或避坑指南。
*   **提议（Proposal）**：如果用户搜索后发现**某门课程尚未被平台收录**，可以通过“提议”功能发起“建课请求”。其他同学们可以对该提议进行“投票/附议”，附议人数达标后系统将正式收录该课程。

## 2. “选课/发现”页面使用方法（首页）
### 2.1 搜索功能
*   在顶部搜索框输入课程名称或教师姓名。
*   **结果展示**：列表会同时显示已收录的课程和尚在提议阶段的课程。
    *   **常规课程卡片（纯白）**：点击进入**课程详情页**，可查看该课所有历史同学的完整评价与打分。
    *   **提议课程卡片（淡粉色渐变）**：卡片右下角会高亮显示当前的**红字附议票数**。点击进入**提议详情页**，不仅能查看发起人的理由，你还可以点击“我要附议”增加票数。

## 3. “我的”页面管理功能
在此页面主要管理你个人发布的“吐槽”与“提议”。

### 3.1 浏览与沉浸式体验
*   页面顶部显示你的总发布数量。
*   **快捷分类**：点击浮动的胶囊按钮（全部/吐槽/提议），可以快速筛选你想查看的内容类型。
*   **吸顶菜单**：当你向下滚动浏览较长的列表时，筛选栏会自动吸附在手机屏幕顶部，不用翻回最上面就能随时切换分类。

### 3.2 互动操作指南
*   **单击卡片**：
    *   点击吐槽卡片 $\rightarrow$ 跳转回该条评价所属的“课程详情主页”。
    *   点击提议卡片 $\rightarrow$ 跳转至“提议动态/详情页”。
*   **长按卡片（高级操作）**：
    找准你需要操作的卡片长按，屏幕底部会弹出选项菜单。
    *   针对**提议**：你可以执行删除。但请注意，如果您选择**“修改”**，系统会提示**“(清空票数)”**，因为修改提议内容对已经投票支持过的同学不公平，所以修改会导致该提议的附议进度重新计算。
    *   针对**吐槽**：常规的修改评价内容或彻底删除自己的评价。

## 4. 快捷发布
在“我的”页面右下角有显眼的红色悬浮 **“+”** 按钮：
*   在“全部”分类下点击，会弹出菜单让你选择是“新增吐槽”还是“新增提议”。
*   在“吐槽”或“提议”特定分类下点击，**会自动跳过菜单选择**，直接进入对应表单的发布流程，节省操作步骤。
